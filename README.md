# Rclone Size Explorer

A simple web app to explore the size of files and directories in a rclone remote.

Hosted on https://rclone-size.surge.sh

## Usage Guide

1. Run `rclone ls remote:path > list.txt` to get the list of files and directories.
2. Select the mode (file/directory/both) and (check/uncheck) the recursive directory size option.
3. Select the `list.txt` file in the web UI. You can also drag and drop the file into the input field.
4. It will display the list of files and directories with their sizes in the web UI.
